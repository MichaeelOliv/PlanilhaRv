# PlanilhaRv

## Sobre

`PlanilhaRv` é um simulador simples em HTML para estimar o valor de RV 2025 para equipes de negócios. A ferramenta permite calcular um valor estimado com base em cargo, tipo de mercado, desembolso mensal, percentual de reembolso e adimplência.

## Funcionalidades

- Seleção de cargo e referência salarial
- Escolha de tipo de mercado
- Entrada de incremento médio, desembolso mensal, reembolso e adimplência
- Cálculo de fator ponderado e bônus de gestão
- Exibição do total estimado de RV em reais

## Como usar

1. Abra o arquivo `index.html` em um navegador moderno.
2. Selecione o cargo desejado.
3. Escolha o tipo de mercado.
4. Informe os valores de incremento médio, desembolso mensal, reembolso e adimplência.
5. O resultado aparece automaticamente na área de resultados.

## Estrutura do projeto

- `index.html` - página principal com o simulador e lógica JavaScript inline.
- `README.md` - documentação do projeto.
